# 斗地主高级策略与心理战术

## 概率与统计决策

### 底牌概率分析
底牌3张中特定牌出现概率：
- 特定单牌：3/41 ≈ 7.3%
- 特定对子成炸：极低，需两张都在底牌
- 补足缺张：取决于手牌缺口

### 剩余牌推算
假设某玩家剩N张牌：
- 最可能牌型：单牌或小对子
- 需防范：炸弹（剩4/6/8张时概率上升）
- 必胜判断：自己牌型能控制对手所有可能出牌

### 期望值决策
计算行动期望值（EV）：
```
EV = 胜率提升 × 收益 - 败率提升 × 代价
```
- 出炸弹的EV：确保本轮控制但暴露实力
- PASS的EV：保留资源但可能失去机会

## 动态策略调整

### 手牌结构分析
优秀手牌结构指标：
1. **完整性**：顺子、连对减少手牌数
2. **控制力**：大牌数量（王、2、A）
3. **灵活性**：三带、飞机等多变牌型
4. **安全度**：炸弹保底

### 局势判断矩阵

| 局势 | 地主策略 | 农民策略 |
|------|----------|----------|
| 开局 | 试探出牌 | 压制大牌 |
| 均势 | 控制节奏 | 配合消耗 |
| 优势 | 加速出完 | 保守防守 |
| 劣势 | 保留炸弹 | 全力压制 |

### 对手建模
根据对手历史行为分类：
- **激进型**：爱叫地主，早用炸弹 → 利用其资源消耗
- **保守型**：不轻易叫地主，保留大牌 → 主动施加压力
- **均衡型**：按标准策略 → 减少失误，等待机会

## 心理战术

### 虚张声势
- 快速出牌营造"必胜"假象
- 保留小牌假装大牌充足
- 用三带消耗看似"调整牌型"

### 诱导出牌
- 出小单诱导对手出2
- 留缺口让对手拆牌
- 假装弱势引诱对手暴露炸弹

### 信息隐藏
- 混合出牌节奏，隐藏手牌结构
- 不急于显示控制力
- 底牌后观察对手反应

## 特定场景策略

### 残局处理（剩牌<5张）

**地主剩3张**：
- 大概率：单+对 或 三张
- 对策：出对子测试，保留大牌

**地主剩2张**：
- 可能：对子 或 两张单牌
- 对策：出非对子牌型，迫使拆分

**地主剩1张**：
- 必是单牌
- 对策：用最小控制牌压制

### 炸弹博弈论

**炸弹展示时机**：
1. 地主展示炸弹 → 农民需权衡是否拆炸弹压制
2. 农民展示炸弹 → 另一农民可放松压制
3. 隐藏炸弹 → 保留意外性和威慑力

**多炸弹博弈**：
- 单炸对单炸：先出者控制节奏
- 双炸对单炸：优势方应主动使用
- 未知炸弹：心理博弈，概率判断

### 顺子与连对决策

**出顺子的时机**：
- 优势：快速减少手牌数
- 劣势：消耗连续牌型，降低灵活性
- 最优：顺子中包含小牌，且大牌仍能控制

**连对 vs 单出**：
- 连对优势：一次出多张，控制节奏
- 连对劣势：需特定牌型，可能被迫拆牌
- 决策：剩余牌数、对手可能压制力

## 实战决策树

### 是否叫地主决策树
```
手牌评估 → 
├─ 大牌数 ≥ 3 (王/2/A) → 继续评估
│   ├─ 有炸弹 → 优势叫地主
│   ├─ 有顺子/连对 → 评估完整性
│   └─ 牌型分散 → 不叫
├─ 大牌数 < 3 → 不叫
└─ 评估底牌风险 → 最终决定
```

### 出牌决策树
```
当前局势 →
├─ 自己能控制 → 出最佳牌型
├─ 队友能赢 → PASS让牌
├─ 地主快赢 → 全力压制
└─ 不明朗 → 保留实力
```

## 常见陷阱与反制

### 陷阱识别
1. **诱导拆牌**：地主出小牌等农民拆大牌
2. **虚假弱势**：地主剩牌少但藏炸弹
3. **配合陷阱**：农民间传递错误信号

### 反制策略
1. **信息验证**：通过出牌测试对手真实实力
2. **保守出牌**：局势不明时保留关键牌
3. **概率优先**：按期望最优决策，不惧虚张
